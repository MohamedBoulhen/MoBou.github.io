# Mohamed Boulhen Portfolio

This is the personal portfolio of Mohamed Boulhen, a 4th-semester Computational Science student at the University of Potsdam. This site showcases various projects I have worked on, highlighting my skills in Java, Python, and C.

## About the Portfolio

This portfolio is built using the "Massively" template by HTML5 UP, a text-heavy, article-oriented design with a large background image and scroll effects powered by Scrollex.

## Features

- **Responsive Design:** Adapts to different screen sizes for optimal viewing on any device.
- **Parallax Background:** Enhanced visual experience with a new parallax implementation.
- **Scroll Effects:** Smooth scrolling powered by Scrollex.
- **Project Showcase:** Highlighting various projects with descriptions and links to GitHub repositories.

## Projects

### Data Analysis - Python
Explore naturalization trends in Germany from 2010 to 2022 using data from the official [source](https://www-genesis.destatis.de/genesis/online). This analysis provides insights into the cultural diversity and integration of naturalized citizens in Germany.
- [View Project](https://github.com/MohamedBoulhen/Naturalisation-Trends-Germany-Insights)

### 2D Game Jump & Run - Java
An exciting Jump and Run adventure game where players navigate through a captivating world, overcoming obstacles, collecting coins, and engaging in battles to progress through levels.
- [View Project](https://github.com/MohamedBoulhen/2D-Game-Jump-Run-Adventure-Java)
